# Short Url Frontend

NextJS Short Url

## Installation

With npm or yarn

```
npm install
```

then run

```
npm run dev
```

Enviroment Variable

```
FNEXT_PUBLIC_APP_URL=http://localhost:3000
NEXT_PUBLIC_API_URL=http://dev.shorturl-backend.local/api
```

## Nice!

NextJS short url frontend is now set up on your homepage.
